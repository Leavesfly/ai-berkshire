## 维度一：财务质量攻击

**核心问题**：利润是真钱还是纸面数字？有没有盈余操纵痕迹？

### 1.1 盈余操纵初筛（M-Score）

```bash
python3 tools/financial_rigor.py m-score \
  --revenue {当年营收} --prev-revenue {上年营收} \
  --receivables {当年应收} --prev-receivables {上年应收} \
  --inventory {当年存货} --prev-inventory {上年存货} \
  --cogs {当年成本} --prev-cogs {上年成本} \
  --sga {当年销售管理费} --prev-sga {上年销售管理费} \
  --total-assets {当年总资产} --prev-total-assets {上年总资产} \
  --current-assets {当年流动资产} --prev-current-assets {上年流动资产} \
  --depreciation {当年折旧} --prev-depreciation {上年折旧} \
  --net-income {当年净利润} --prev-net-income {上年净利润} \
  --cfo {当年经营现金流} --prev-cfo {上年经营现金流} \
  --market-cap {市值} --total-debt {总负债}
```

- M-Score > -1.78 → 操纵嫌疑（🔴）
- -2.22 < M-Score < -1.78 → 灰色地带（🟡）
- M-Score < -2.22 → 正常（⚪）

### 1.2 现金流背离

- 经营现金流 / 净利润 < 80% → 利润含金量不足
- 连续 2 年以上背离 → 系统性问题
- 自由现金流为负但利润为正 → 资本开支吞噬利润

### 1.3 应计质量

```bash
python3 tools/financial_rigor.py accruals \
  --net-income {净利润} --cfo {经营现金流} --cfi {投资现金流} \
  --total-assets {总资产} --prev-total-assets {上年总资产}
```

- 应计比率 > 15% → 利润过度依赖非现金项目（🔴）

### 1.4 资产负债表恶化信号

- 应收账款增速 > 营收增速 → 赊销冲量
- 存货增速 > 营收增速 → 滞销积压
- 商誉 / 净资产 > 30% → 减值地雷
- 短期借款激增 + 经营现金流恶化 → 流动性危机前兆

### 1.5 财务困境（Altman-Z）

```bash
python3 tools/financial_rigor.py altman-z --model em \
  --total-assets {总资产} --retained-earnings {留存收益} \
  --ebit {息税前利润} --market-cap {市值} \
  --total-liabilities {总负债} --revenue {营收} \
  --working-capital {营运资本} --current-liabilities {流动负债}
```

## 维度二：护城河侵蚀

**核心问题**：护城河是在变窄还是已经没了？

### 2.1 定价权丧失

- 毛利率连续 2 年下滑 → 竞争加剧/成本转嫁失败
- 提价后销量显著下降 → 品牌溢价消失
- 促销/补贴占营收比上升 → 靠烧钱维持份额

### 2.2 客户流失

- 客户集中度上升（前 5 大客户占比 > 50%）→ 议价权转移
- 用户增长放缓 + ARPU 下降 → 飞轮停转
- 大客户自建替代方案 → 转换成本归零

### 2.3 竞争格局恶化

- 新进入者以更低成本/更好体验切入
- 行业 CR3/CR5 下降 → 集中度分散化
- 价格战爆发 → 差异化消失

### 2.4 技术替代

- 核心技术被开源/标准化 → 壁垒归零
- 下一代技术路线绕过当前方案
- 研发投入 / 营收比持续低于竞争对手

## 维度三：管理层红旗

**核心问题**：管理层在隐瞒什么？在跑吗？

### 3.1 内部人减持信号

```bash
python3 tools/insider_trading.py recent {代码} --days 365
```

- 多名高管同期非计划性减持 → 强警告（🔴）
- CEO/CFO 减持比例 > 持股 30% → 信心不足
- 减持时点恰在利好公告前 → 信息不对称嫌疑
- 注意：单笔减持≠看空（缴税/行权/流动性），需综合判断

### 3.2 承诺不兑现

- 搜索过去 2-3 年管理层公开承诺（业绩指引/战略目标）
- 逐条对比实际结果
- 连续 miss 且无合理解释 → 诚信问题

### 3.3 激进会计政策

- 收入确认政策变更（提前确认）
- 折旧/摊销年限延长（美化利润）
- 频繁变更审计师
- 非经常性损益占比 > 20%

### 3.4 关联交易与利益输送

- 大股东/关联方交易金额异常
- 高管薪酬与业绩脱钩
- 频繁并购 + 高额商誉 → 利益输送嫌疑

## 维度四：估值泡沫

**核心问题**：市场在为什么幻觉付费？

### 4.1 隐含增长不可持续

```bash
python3 tools/financial_rigor.py reverse-dcf \
  --market-cap {市值} --fcf {自由现金流} --wacc {贴现率} --terminal-growth {永续增长}
```

- 隐含增长率 > 行业 TAM 增速 → 市场假设不现实
- 隐含增长率 > 公司历史最高增速 → 需要"奇迹"

### 4.2 历史极端分位

```bash
python3 tools/financial_rigor.py valuation-percentile \
  --price {价格} --metric pe --years 10
```

- PE/PS 处于 10 年 > 90% 分位 → 极端乐观定价
- 与同业对比溢价 > 50% 且无基本面支撑 → 泡沫嫌疑

### 4.3 叙事驱动 vs 数据驱动

- 估值逻辑依赖"未来某天盈利"而非当前现金流
- 市场用 TAM（总可寻址市场）而非 SAM/SOM 定价
- 类比估值（"下一个 XX"）替代基本面分析

## 维度五：终局风险

**核心问题**：5-10年后这门生意还在吗？

### 5.1 技术范式转移

- 当前技术路线是否处于 S 曲线顶部？
- 替代技术是否已过实验室阶段？
- 历史类比：柯达/诺基亚/传统出租车

### 5.2 监管颠覆

- 行业是否面临结构性监管收紧（反垄断/数据安全/环保）？
- 政策补贴退坡 → 伪需求暴露
- 准入门槛降低 → 供给过剩

### 5.3 需求消失

- 核心用户群是否在萎缩？
- 消费习惯/人口结构变化是否不可逆？
- "解决方案在找问题"还是"问题在找解决方案"？

### 5.4 文明趋势逆转（李录视角）

- 站在 20 年后回看，这家公司是"标准石油"还是"3Com"？
- 是否处于被文明趋势淘汰的一侧？
- 永久性资本损失的最可能路径是什么？

## 攻击强度判定标准

| 等级 | 标准 | 含义 |
|------|------|------|
| 🔴 强 | ≥2 条独立数据证据 + 逻辑链完整 | 足以单独动摇投资论点 |
| 🟡 中 | 1 条数据证据 + 合理推断 | 需要持续跟踪，暂不致命 |
| ⚪ 弱 | 仅有逻辑推演，缺乏数据支撑 | 理论风险，当前无实证 |

## 输出要求

- 每个维度必须给出攻击强度评级（🔴/🟡/⚪）+ 置信度（高/中/低）
- 数据必须来自工具输出或双源验证，标注来源
- 找不到强论据时如实标注"该维度未发现有效攻击点"
- 禁止用"可能""也许""不排除"填充——要么有证据，要么承认没有
